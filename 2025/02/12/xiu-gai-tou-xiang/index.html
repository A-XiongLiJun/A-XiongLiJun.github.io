修改头像

1 图片上传预览

  1. 在页面中添加 input 上传框

    <input type="file" hidden ref="file" >

  2. 点击头像显示上传获取input 对象

    <!-- 个人信息 -->
<van-cell title="头像" is-link @click="$refs.file.click()">
  <van-image class="avatar" fit="cover" round :src="user.photo" />
</van-cell>

  3. 给 input 上传框绑定 onChange 事件，触发事件处理程序 onFileChange

    <input type="file" hidden ref="file" @change="onFileChange" />

    // 更新头像
onFileChange() {
  // 获取文件对象
  const file = this.$refs.file.files[0]
  // 基于文章对象获取到 blob 数据
  const data = window.URL.createObjectURL(file)
  console.log(data)
}

  4. 刷新页面，查看是否获取到 blob 数据

  2. 在 data 中声明 isUpdatePhotoShow 属性，控制弹框的显示状态

    data() {
  return {
    // ...
    isUpdatePhotoShow: false // 控制修改头像的弹框
  }
}

  3. 在获取上传图片的内容后，让弹框显示

    // 更新头像
onFileChange() {
  // ...

  // 展示预览图片弹出层
  this.isUpdatePhotoShow = true
}

  4. 刷新页面，查看弹框是否展示

<style scoped lang="less"></style>

    • 导入组件

  3. 将父组件中获取到的图片信息赋值给 data 中的 img 属性并传递到 update-photo.vue 组件中

    data() {
  return {
    // ...
    img: null // 预览的图片
  }
}

    // 更新头像
onFileChange() {
  // 获取文件对象
  const file = this.$refs.file.files[0]
  // 基于文章对象获取到 blob 数据
  this.img = window.URL.createObjectURL(file)

  // 展示预览图片弹出层
  this.isUpdatePhotoShow = true

  // file-input 如果选了同一个文件不会触发 change 事件
  // 解决办法就是每次使用完毕，把它的 value 清空
  this.$refs.file.value = ''
}

    <!-- 编辑头像 -->
<van-popup v-model="isUpdatePhotoShow" position="bottom" style="height: 100%;">
  <update-photo v-if="isUpdatePhotoShow" :img="img" />
</van-popup>

  4. 在子组件中使用 props 接收到传递的数据，并渲染到组件中

    export default {
  name: 'UpdatePhoto',
  props: {
    img: {
      type: [String, Object],
      required: true,
    },
  },
}

    <template>
  <div class="update-photo">
    <img class="img" :src="img" />
  </div>
</template>

  1. 如果是基于服务端的裁切，则使用：getData 方法，该方法得到裁切的区域参数

  2. 如果是纯客户端的图片裁切，则使用：getCroppedCanvas 方法，该方法得到裁切之后的图片对象（类似于URL.createObjectURL 方法得到的文件对象）

7 保存更新

  1. 封装发送 ajax 请求的方法

    // 更新用户照片资料
export const updateUserPhoto = data => {
  return request({
    method: 'PATCH',
    url: '/app/v1_0/user/photo',
    data,
  })
}

  2. 在 update-photo.vue 组件中导入封装的接口方法

    import { updateUserPhoto } from '@/api/user'

  3. 点击完成后发送接口请求

    async updateUserPhoto(blob) {
  this.$toast.loading({
    message: '保存中...',
    forbidClick: true, // 禁止背景点击
    duration: 0 // 持续展示
  })
  try {
    // 错误的用法
    // 如果接口要求 Content-Type 是 application/json
    // 则传递普通 JavaScript 对象
    // updateUserPhoto({
    //   photo: blob
    // })

    // 如果接口要求 Content-Type 是 multipart/form-data
    // 则你必须传递 FormData 对象
    const formData = new FormData()
    formData.append('photo', blob)

    const { data } = await updateUserPhoto(formData)

    // 关闭弹出层
    this.$emit('close')

    // 更新视图
    this.$emit('update-photo', data.data.photo)

    // 提示成功
    this.$toast.success('更新成功')
  } catch (err) {
    this.$toast.fail('更新失败')
  }
}

  4. 在父组件中接收 $emit 触发的事件

<!-- 编辑头像 -->
<van-popup v-model="isUpdatePhotoShow" position="bottom" style="height: 100%;">
  <update-photo
    v-if="isUpdatePhotoShow"
    :img="img"
    @close="isUpdatePhotoShow = false"
    @update-photo="user.photo = $event"
  />
</van-popup>
